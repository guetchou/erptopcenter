# TopCenter HR

Module **à la carte** : extensions RH spécifiques TopCenter.

- Dépend de **core** (erpnext, hrms, topcenter_core, topcenter_branding).
- Prévu pour accueillir le refactor / évolution de **congo_hrms** (congés, paie, QR, workflows, etc.).

Installation : `bench --site <site> install-app topcenter_hr` ou via `install.sh --modules core,hr`.
